# Hoe kan ik mijn gegevens wijzigen?

> Bron: https://help.keurigonline.nl/article/35-hoe-kan-ik-mijn-gegevens-wijzigen

Je kan je adres- en contactgegevens eenvoudig wijzigen in [Mijn KeurigOnline](https://mijn.keurigonline.nl/).

Indien je je wachtwoord bent vergeten kan je deze makkelijk en snel resetten via: [Wachtwoord vergeten?](https://mijn.keurigonline.nl/wachtwoord-vergeten)

Weet je het klantnummer voor je diensten niet meer? Vraag deze dan op via: [Klantnummer vergeten?](https://mijn.keurigonline.nl/klantnummer-vergeten)
