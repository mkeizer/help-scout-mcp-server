# Hoe kan ik betalen per automatische incasso?

> Bron: https://help.keurigonline.nl/article/38-hoe-kan-ik-betalen-per-automatische-incasso

Als je al een bestelling heeft geplaatst kun je in Mijn KeurigOnline onder "Mijn Gegevens" de betaalmethode wijzigen naar Automatische Incasso.

Bij een nieuwe bestelling kan dit opgegeven worden bij het afronden van de bestelling.

Let wel op! Bij een nieuwe bestelling met Automatische incasso wordt er een handmatige check gedaan. De levering van je diensten kan dus iets langer op zich laten wachten.
