# Hoe kan ik mijn facturen betalen?

> Bron: https://help.keurigonline.nl/article/31-hoe-kan-ik-mijn-facturen-betalen

Het meest eenvoudig is om de betaling te doen via iDeal. Als we een factuur sturen per e-mail staat er altijd een linkje bij om de betaling via iDeal te starten.

Eventueel kan je ook handmatig een betaling via iDeal starten via onze [betaalpagina](https://betalen.keurigonline.nl/).

Handmatig overboeken kan uiteraard ook. Maak de facturen over naar NL03ABNA0618697837 ten name van KeurigOnline B.V. Vermeld in de omschrijving duidelijk het klantnummer en de facturen waar de betaling voor is!
