# Waarom kan ik mijn domeinnaam niet bereiken?

> Bron: https://help.keurigonline.nl/article/63-waarom-kan-ik-mijn-domeinnaam-niet-bereiken

Als de domeinnaam niet bereikbaar lijkt te zijn, zijn er een paar dingen die je kan controleren voordat je contact met ons op neemt.

- Werkt de website op meerdere internetverbindingen niet? - Is de site ook offline volgens een website zoals &#39;[downforeveryoneorjustme.com](https://downforeveryoneorjustme.com)&#39;?- Is mijn website offline naar sinds werkzaamheden die ik zelf heb uitgevoerd?- Heb ik nog genoeg schijfruimte in mijn hosting account? Bij een tekort kunnen er problemen optreden.In alle andere gevallen kan je contact met ons opnemen met onze helpdesk. Vaak betreft het dan een IP-blokkade. Ga alvast naar [https://checkhost.nl](https://checkhost.nl) zodat je snel het IP-adres door kan geven aan een van onze medewerkers.
