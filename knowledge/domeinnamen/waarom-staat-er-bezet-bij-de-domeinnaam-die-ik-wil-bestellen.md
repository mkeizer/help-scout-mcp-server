# Waarom staat er 'bezet' bij de domeinnaam die ik wil bestellen?

> Bron: https://help.keurigonline.nl/article/101-waarom-staat-er-bezet-bij-de-domeinnaam-die-ik-wil-bestellen

Indien de ingevoerde domeinnaam als status bezet aangeeft, betekent dat de domeinnaam reeds geregistreerd is, ben je zelf de eigenaar van de domeinnaam? Dan kan je via de optie &#39;verhuizen&#39; de domeinnaam verhuizen naar KeurigOnline.

Om de domeinnaam te kunnen verhuizen heb je een EPP code nodig, deze code kan je bij de huidige provider opvragen. De EPP code kan je na afloop van de bestelling invoeren via de optie &#39;verhuizingen&#39; in mijn account.

De EPP-code staan beter bekend als de verhuiscode of token. Dit is een code waarmee het mogelijk is  gemaakt om een domeinnaam te verhuizen naar een andere hostingprovider.
