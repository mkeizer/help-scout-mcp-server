# Hoe voeg ik een domeinnaam toe?

> Bron: https://help.keurigonline.nl/article/42-hoe-voeg-ik-een-domeinnaam-toe

Om een nieuwe domeinnaam aan te maken in je hostingpakket moet je zijn ingelogd op DirectAdmin. De DirectAdmin-omgeving is altijd te benaderen via; https://jedomeinnaam.nl:2222

Zodra je bent ingelogd op DirectAdmin krijg je een navigatiebalk te zien. Klik op Accountbeheer.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db43bd593d719515c97b5/file-WYWxS7QHNg.png)

Selecteer in de lijst vervolgens &#39;Domein instellingen&#39;

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db4671f3fa9421e286ec5/file-SwgGX1V0pX.png)

Om een nieuwe domeinnaam toe te voegen binnen je hostingpakket selecteer je bovenin &#39;Nieuwe toevoegen&#39;

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db487d593d719515c97b6/file-xHDFFYGciH.png)

Vul hier vervolgens je domeinnaam in, en eventueel specifieke waarden in voor die domeinnaam.

Klik vervolgens op &#39;Aanmaken&#39;. Je hebt de domeinnaam nu succesvol toegevoegd.

Let op! Een domeinnaam toevoegen aan je DirectAdmin pakket is niet hetzelfde als het registreren van een domeinnaam. Als wij deze voor je moeten registreren kan je deze [bestellen](https://www.keurigonline.nl/bestellen/registreren) via onze website.
