# KeurigOnline Begrippenlijst

> Bron: https://www.keurigonline.nl/begrippenlijst
> Laatste sync: 2026-03-06

**2FA**: 2FA (tweefactorauthenticatie) is een beveiligingsmethode die naast je wachtwoord een tweede verificatie vereist.

**AAAA-record**: Een AAAA-record koppelt een domeinnaam aan een IPv6-adres.

**Alias**: Een alias is een extra naam gekoppeld aan een domein of e-mailadres.

**Anycast**: Anycast is een netwerktechniek waarbij meerdere servers hetzelfde IP-adres delen en verkeer naar de dichtstbijzijnde server wordt gerouteerd.

**Apache**: Apache HTTP Server is een open source webserver ontwikkeld door de Apache Software Foundation.

**API**: Een API (Application Programming Interface) is een set regels en protocollen waarmee softwareapplicaties met elkaar kunnen communiceren en gegevens uitwisselen.

**A-record**: Een A-record (Address Record) koppelt een domeinnaam aan een IPv4-adres. Het is het meest gebruikte DNS-record voor websites.

**Autoresponder**: Een autoresponder verstuurt automatisch antwoorden op ontvangen e-mails. Vakantieberichten werken automatisch binnen vooraf ingestelde data.

**Autoritatieve nameserver**: Een autoritatieve nameserver bevat de officiële DNS-records voor een domein en geeft definitieve antwoorden.

**AWstats**: AWstats is een open-source tool voor het analyseren van webserver logbestanden en genereren van statistieken.

**Backup**: Een backup is een reservekopie van digitale gegevens die je kunt terugzetten bij dataverlies of storingen.

**Bandwidth**: Bandwidth is de maximale hoeveelheid data die per tijdseenheid over een netwerkverbinding kan worden verstuurd.

**BIMI record**: BIMI (Brand Indicators for Message Identification) toont je merklogo naast geauthenticeerde emails in de inbox.

**Blacklist**: Een blacklist is een lijst met geblokkeerde IP-adressen, domeinnamen of e-mailadressen die als spam of onveilig worden beschouwd.

**Bounce rate**: Bounce rate is het percentage bezoekers dat je website verlaat zonder verdere actie.

**Brotli**: Brotli is een compressie-algoritme ontwikkeld door Google dat webinhoud effectiever comprimeert dan gzip.

**Brute force-aanval**: Een brute force-aanval is een methode waarbij een aanvaller systematisch alle mogelijke combinaties probeert om toegang te krijgen tot een systeem.

**Cache**: Een cache is een tijdelijke opslaglaag die veelgebruikte gegevens bewaart voor snellere toegang en betere prestaties.

**Cache poisoning**: Cache poisoning is een aanval waarbij vervalste DNS-informatie wordt opgeslagen om verkeer om te leiden.

**Canonical tag**: Een canonical tag vertelt zoekmachines welke versie van een pagina de &#039;originele&#039; is.

**Captcha**: Captcha is een beveiligingstest die onderscheid maakt tussen mensen en geautomatiseerde programma&#039;s (bots).

**CDN**: Een CDN (Content Delivery Network) is een wereldwijd netwerk van servers dat content sneller bij gebruikers aflevert.

**CHMOD**: CHMOD is een Unix-commando dat bestandspermissies wijzigt, zoals lees-, schrijf- en uitvoerrechten voor gebruikers, groepen en anderen.

**CloudLinux**: CloudLinux is een gespecialiseerd Linux-besturingssysteem voor webhostingproviders dat elke gebruiker isoleert in een eigen container.

**CLS**: CLS meet hoeveel je pagina-layout verspringt tijdens het laden.

**CMS**: Een CMS is een platform om webcontent eenvoudig te creëren, beheren en publiceren zonder te coderen.

**CNAME-record**: Een DNS-record dat een domeinnaam als alias laat verwijzen naar een andere domeinnaam.

**Composer**: Composer is een dependency manager voor PHP die het beheer van externe bibliotheken en afhankelijkheden in projecten automatiseert.

**Core Web Vitals**: Core Web Vitals zijn Google&#039;s belangrijkste metrics voor website-ervaring en snelheid.

**Crawl budget**: Crawl budget is het aantal pagina&#039;s dat Google per tijdseenheid van je website crawlt.

**Critical CSS**: Critical CSS is de minimale CSS die nodig is om content boven de vouw te stylen.

**Cronjobs**: Een cronjob is een geplande taak in Unix-achtige systemen die automatisch op specifieke tijden wordt uitgevoerd.

**CSS**: CSS (Cascading Style Sheets) is een opmaaktaal voor het vormgeven en stylen van webpagina&#039;s en documenten.

**Database**: Een database is een georganiseerde verzameling gegevens die elektronisch wordt opgeslagen en beheerd.

**DDoS-aanval**: Een cyberaanval waarbij een website of server onbereikbaar wordt gemaakt door deze te overspoelen met verkeer.

**DirectAdmin**: DirectAdmin is een webhosting control panel ontwikkeld door JBMC Software als kostenefficiënt en gebruiksvriendelijk alternatief voor cPanel.

**DKIM**: DKIM beveiligt e-mails met een digitale handtekening om spam tegen te gaan.

**DMARC**: DMARC is een e-mailbeveiligingsprotocol dat domeinen beschermt tegen phishing en spoofing via SPF en DKIM.

**DNS**: Het Domain Name System (DNS) vertaalt domeinnamen naar IP-adressen, zodat websites makkelijk bereikbaar zijn.

**DNS resolver**: Een DNS resolver is een server die domeinnamen vertaalt naar IP-adressen voor je internetverbinding.

**DNSSEC**: DNSSEC (DNS Security Extensions) beveiligt DNS-informatie door deze cryptografisch te ondertekenen.

**Docker**: Docker faciliteert het verpakken en draaien van applicaties in gestandaardiseerde, geïsoleerde containers.

**DoH**: DNS over HTTPS (DoH) versleutelt DNS-verzoeken via het HTTPS-protocol, waardoor je surfgedrag privé blijft.

**Domeinnaam**: Een domeinnaam is een uniek webadres gekoppeld aan je website en e-mail, zoals voorbeeld.nl, bedrijf.com of startup.ai.

**Domeinverhuizing**: Een domeinverhuizing is het overzetten van een domeinnaam naar een andere registrar of hostingprovider.

**DoQ**: DNS over QUIC (DoQ) combineert versleuteling met de snelheid van het QUIC-protocol voor moderne DNS-privacy.

**DoT**: DNS over TLS (DoT) versleutelt DNS-verkeer via TLS op poort 853, waardoor afluisteren wordt voorkomen.

**EDNS**: EDNS (Extension Mechanisms for DNS) is een protocoluitbreiding die de 512-byte limiet van DNS opheft voor moderne functionaliteit.

**E-mailadres**: Een e-mailadres is een uniek digitaal adres in het formaat naam@domein.nl voor het versturen en ontvangen van elektronische berichten.

**Email authenticatie**: Email authenticatie is een verzameling DNS-protocollen (SPF, DKIM, DMARC) die bewijzen dat een email daadwerkelijk van het geclaimde domein komt.

**Email deliverability**: Email deliverability is de mate waarin je emails succesvol in de inbox terechtkomen in plaats van spam.

**E-mail forwarding**: E-mail forwarding stuurt inkomende e-mails automatisch door naar een ander e-mailadres.

**Email spoofing**: Email spoofing is het vervalsen van de afzender van een email om te doen alsof deze van een ander domein komt.

**Encryptie**: Encryptie is het versleutelen van gegevens zodat alleen bevoegden ze kunnen lezen.

**Firewall**: Een firewall is een beveiligingssysteem dat netwerkverkeer filtert en ongeautoriseerde toegang tot netwerken en servers blokkeert.

**Foutlog**: Een foutlog registreert fouten en problemen op je website voor probleemoplossing.

**FTP**: FTP (File Transfer Protocol) is een protocol voor het overdragen van bestanden tussen computers via een netwerk.

**FTP account**: Een FTP-account is een gebruikersaccount met inloggegevens voor het uploaden en beheren van bestanden op een webserver via het File Transfer Protocol.

**GeoDNS**: GeoDNS geeft verschillende IP-adressen terug op basis van de geografische locatie van de gebruiker.

**Git**: Git is een gedistribueerd versiebeheersysteem voor het bijhouden van wijzigingen in broncode tijdens softwareontwikkeling.

**Github**: GitHub is een cloudplatform voor versiebeheer en samenwerking aan softwareprojecten met Git.

**Glue record**: Een glue record is een DNS-record met het IP-adres van een nameserver om circulaire afhankelijkheden op te lossen.

**GZIP-compressie**: GZIP verkleint websitebestanden tijdens verzending voor snellere laadtijden.

**High-availability**: High-availability (HA) garandeert continue beschikbaarheid van systemen met minimale downtime.

**HSTS**: HSTS forceert browsers om alleen via HTTPS te verbinden met je website.

**htaccess**: Een .htaccess bestand is een configuratiebestand voor Apache webservers waarmee je per directory instellingen kunt wijzigen zonder toegang tot de hoofdconfiguratie.

**HTML**: HTML is de programmeertaal voor het maken van webpagina’s.

**htpasswd**: .htpasswd is een bestand voor wachtwoordbeveiliging van webpagina’s of mappen.

**HTTP**: HTTP (HyperText Transfer Protocol) is het protocol dat de communicatie tussen webclients en servers mogelijk maakt.

**HTTP-statuscodes**: HTTP-statuscodes zijn driecijferige codes die aangeven of een webverzoek succesvol was of niet.

**IMAP**: IMAP (Internet Message Access Protocol) is een protocol voor het ontvangen en beheren van e-mail op een externe server.

**Imunify360**: Imunify360 is een uitgebreide beveiligingssuite voor Linux webservers met AI-aangedreven malware scanning.

**INP**: INP (Interaction to Next Paint) is een Core Web Vital die de responstijd van je website op gebruikersinteracties meet.

**Installatron**: Installatron is een one-click installer voor het automatisch installeren van webapplicaties zoals WordPress.

**IPv4**: IPv4 is het 32-bit adresseringssysteem dat het grootste deel van het internet nog steeds aandrijft.

**IPv6**: IPv6 is de nieuwste versie van het internetprotocol dat IPv4 opvolgt met een enorme adresruimte van 128-bit.

**ISP**: Een ISP (Internet Service Provider) is de aanbieder van jouw internetverbinding.

**JavaScript**: JavaScript is een programmeertaal die wordt gebruikt om interactieve en dynamische websites te maken.

**JetBackup**: JetBackup is een backup-software voor webhostingomgevingen die incrementele back-ups en self-service herstel biedt.

**Joomla**: Joomla is een gratis, open-source content management systeem (CMS) voor het bouwen van websites.

**JSON**: JSON is een lichtgewicht dataformaat voor het uitwisselen van gegevens tussen systemen.

**Kernel**: Een kernel is de kern van een besturingssysteem die de communicatie tussen hardware en software beheert.

**Laravel**: Laravel is een open-source PHP-framework voor het bouwen van webapplicaties volgens het MVC-patroon.

**Lazy loading**: Lazy loading is een techniek waarbij afbeeldingen en content pas worden geladen wanneer ze in beeld komen.

**LCP**: LCP (Largest Contentful Paint) meet de laadtijd van het grootste zichtbare element op een webpagina.

**Lighthouse**: Lighthouse is een open-source tool van Google die webpagina&#039;s auditet op performance, toegankelijkheid en SEO.

**LiteSpeed**: LiteSpeed is een snelle webserver die tot 6 keer sneller is dan Apache en gebruikmaakt van een efficiënte event-driven architectuur.

**LiteSpeed Cache**: LiteSpeed Cache is een server-level cachingoplossing die dynamische pagina&#039;s opslaat als statische HTML-bestanden.

**Load balancing**: Load balancing verdeelt verkeer over meerdere servers voor betere prestaties en betrouwbaarheid.

**Magento**: Magento is een krachtig open-source e-commerce platform voor professionele webshops.

**Malware-scan**: Een malware-scan controleert bestanden en code op virussen en kwaadaardige software.

**Metatags**: Metatags zijn HTML-elementen die metadata over een webpagina verstrekken aan zoekmachines en browsers.

**Minification**: Minification verkleint code door onnodige tekens te verwijderen zonder functionaliteit te veranderen.

**MTA-STS**: MTA-STS dwingt TLS-versleuteling af voor email transport en voorkomt downgrade-aanvallen op SMTP-verbindingen.

**MX-record**: Een MX-record is een DNS-record dat aangeeft welke mailserver verantwoordelijk is voor het ontvangen van e-mail voor een domein.

**MySQL**: MySQL is een open-source relationeel databasebeheersysteem (RDBMS) dat gegevens opslaat in tabellen.

**Nameservers**: Nameservers zijn DNS-servers die domeinnamen omzetten naar IP-adressen en internetverkeer naar de juiste servers leiden.

**Negative caching**: Negative caching slaat &quot;niet gevonden&quot; DNS-antwoorden (NXDOMAIN) tijdelijk op om herhaalde queries te voorkomen.

**Nginx**: Nginx is een open-source webserver, reverse proxy en load balancer die veel gebruikt wordt voor hoogperformante websites.

**NVMe**: NVMe is ultrasnelle SSD-opslag die websites tot 6x sneller maakt dan traditionele opslag.

**NXDOMAIN**: NXDOMAIN is een DNS-antwoord dat aangeeft dat het opgevraagde domein niet bestaat.

**Oblivious DoH**: Oblivious DoH (ODoH) routeert DNS-queries via een proxy, zodat de provider je query ziet maar niet je IP-adres.

**osCommerce**: osCommerce is gratis software om eenvoudig je eigen webshop te beheren.

**PageSpeed**: Meetinstrument van Google voor websiteprestaties en optimalisatiesuggesties.

**PageSpeed Insights**: PageSpeed Insights is Google&#039;s gratis tool om website-snelheid en Core Web Vitals te testen.

**Phishing**: Phishing is een cyberaanval waarbij criminelen zich voordoen als betrouwbare organisaties om gevoelige informatie te stelen.

**PHP**: PHP (Hypertext Preprocessor) is een populaire open-source scripttaal die speciaal is ontworpen voor webontwikkeling en server-side programmering.

**POP3**: POP3 (Post Office Protocol versie 3) is een e-mailprotocol waarmee je berichten van een mailserver downloadt en lokaal opslaat.

**PTR-record**: Een PTR-record vertaalt een IP-adres naar een domeinnaam (reverse DNS) en is cruciaal voor e-mailbeveiliging.

**Quota**: Een quota is een limiet op de schijfruimte of het aantal bestanden dat je mag gebruiken binnen je hostingpakket.

**Redirect**: Een redirect stuurt bezoekers automatisch door naar een andere webpagina.

**Register Globals**: Register Globals is een verouderde en onveilige PHP-instelling die standaard uitgeschakeld is.

**Render-blocking**: Render-blocking resources zijn bestanden die de browser eerst moet laden voordat de pagina zichtbaar wordt.

**Reseller hosting**: Reseller hosting stelt je in staat webhostingdiensten onder je eigen merknaam aan te bieden zonder eigen serverinfrastructuur.

**Reverse proxy**: Een reverse proxy is een server die verzoeken ontvangt en doorstuurt naar backend-servers.

**Root server**: Root servers zijn de 13 server-clusters aan de top van de DNS-hiërarchie die verwijzen naar TLD-nameservers.

**Roundcube**: Roundcube is een open-source webmail-client met een moderne interface en volledige IMAP-ondersteuning.

**Round-robin DNS**: Round-robin DNS verdeelt verkeer over meerdere servers door de volgorde van A-records te roteren bij elke query.

**Security Headers**: HTTP-headers die je website extra beschermen.

**Shared hosting**: Shared hosting is een hostingvorm waarbij meerdere websites dezelfde serverresources delen.

**SIDN**: SIDN beheert als officiële registry alle .nl-domeinnamen in Nederland sinds 1996.

**Sitemap**: Een sitemap is een bestand dat zoekmachines helpt alle pagina&#039;s van je website te vinden.

**SOA record**: Het SOA (Start of Authority) record bevat beheerinformatie over een DNS-zone, zoals serial number en TTL-waarden.

**Spamfilter**: Een spamfilter blokkeert ongewenste e-mails. Bij ons kies je tussen interne (SpamAssassin) en externe (SpamExperts) spamfilters.

**SPF flattening**: SPF flattening vervangt include-statements door directe IP-adressen om onder de 10 DNS lookup limiet te blijven.

**SPF-record**: Een SPF-record is een DNS-record dat bepaalt welke mailservers e-mail mogen versturen namens een domein.

**Split-horizon DNS**: Split-horizon DNS geeft verschillende antwoorden afhankelijk van of een query intern of extern komt.

**SSD**: SSD-opslag is snelle solid-state opslag die je website aanzienlijk versnelt.

**SSH**: SSH (Secure Shell) is een beveiligingsprotocol voor versleutelde externe toegang tot computers en servers.

**SSH-key**: Een SSH-key is een cryptografisch sleutelpaar dat veilige authenticatie mogelijk maakt tussen een client en server zonder wachtwoorden.

**SSL**: SSL (Secure Sockets Layer) is een beveiligingsprotocol voor versleutelde internetverbindingen.

**Staging-omgeving**: Een staging-omgeving is een testomgeving die de productieomgeving nabootst om wijzigingen te testen.

**Statistieken**: Statistieken laten zien hoe bezoekers je website gebruiken.

**Subdomein**: Een subdomein is een extra voorvoegsel voor je domeinnaam, zoals blog.voorbeeld.nl of shop.voorbeeld.nl.

**suPHP**: suPHP is een Apache-module die PHP-scripts uitvoert met de rechten van de eigenaar voor extra beveiliging.

**TLD nameserver**: Een TLD nameserver beheert DNS-informatie voor een topleveldomein zoals .nl, .com of .be.

**TLS**: TLS is het standaardprotocol voor versleutelde, betrouwbare communicatie tussen client en server.

**TLSA record**: TLSA records publiceren certificaatinformatie via DNS voor DANE-authenticatie, onafhankelijk van Certificate Authorities.

**TLS-RPT**: TLS-RPT is een DNS record voor het ontvangen van rapporten over TLS-fouten bij email transport.

**TTFB**: TTFB (Time to First Byte) meet de tijd tussen een HTTP-verzoek en de eerste byte van het serverantwoord.

**TTL**: TTL bepaalt hoe lang DNS-informatie of cache-data geldig blijft voordat deze ververst wordt.

**TXT-record**: Een TXT-record bevat tekstinformatie in DNS, vaak gebruikt voor domeinverificatie en e-mailbeveiliging.

**Uptime**: Uptime is de tijd dat een server of website operationeel en bereikbaar is, uitgedrukt als percentage.

**Vakantiebericht**: Een vakantiebericht is een tijdelijke automatische e-mail waarin je aangeeft wanneer je afwezig bent en wanneer je weer reageert.

**VMC certificaat**: Een VMC (Verified Mark Certificate) bewijst eigenaarschap van een handelsmerk en is vereist voor BIMI in Gmail.

**VPS**: Een VPS is een eigen virtuele server met meer vrijheid en flexibiliteit dan shared hosting.

**Wachtwoordmanager**: Een wachtwoordmanager is software die al je wachtwoorden veilig en versleuteld opslaat.

**Waterfall chart**: Een waterfall chart visualiseert de laadvolgorde en timing van alle resources op je pagina.

**Webmail**: E-mail bekijken en beheren via een webbrowser zonder aparte software, met toegang vanaf elk apparaat met internetverbinding.

**WebP**: WebP is een modern afbeeldingsformaat dat 25-35% kleiner is dan JPEG en PNG.

**Websocket**: Real-time verbinding tussen server en browser.

**WHOIS**: WHOIS is een protocol en database waarmee je kunt opzoeken wie de eigenaar is van een domeinnaam of IP-adres.

**Wildcard-domein**: Een wildcard-domein is een DNS-record met een asterisk (*) die alle niet-gedefinieerde subdomeinen opvangt.

**WooCommerce**: WooCommerce is een gratis e-commerce plugin voor WordPress waarmee je een webshop kunt bouwen en beheren.

**WordPress**: WordPress is een open-source contentmanagementsysteem (CMS) geschreven in PHP, waarmee je eenvoudig websites en blogs kunt bouwen en beheren.

**WP-CLI**: WP-CLI is de command line interface voor WordPress waarmee ontwikkelaars WordPress beheren via de terminal.

**WWW**: WWW (World Wide Web) is een systeem van onderling verbonden webpagina&#039;s die via internet toegankelijk zijn.

**XML**: XML is een opmaaktaal die gebruikt wordt om gestructureerde data uit te wisselen tussen systemen.

**XSS**: XSS is een veelvoorkomende aanval waarbij scripts worden geïnjecteerd in websites om gebruikersdata te stelen.

**YAML**: YAML is een leesbare datastructuurtaal, vaak gebruikt voor configuratiebestanden zoals in CI/CD-pijplijnen.

**ZIP**: ZIP is een bestandsformaat voor het comprimeren van meerdere bestanden tot één kleiner archief.

**Zonefile**: Een zonefile is een tekstbestand met alle DNS-records van een domein, zoals A, MX, CNAME en NS.

**Zone transfer**: Een zone transfer (AXFR/IXFR) kopieert DNS-records van een primary naar secondary nameserver voor synchronisatie.
