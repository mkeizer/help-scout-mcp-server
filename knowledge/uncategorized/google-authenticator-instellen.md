# Google Authenticator instellen voor Mijn KeurigOnline

> Bron: https://help.keurigonline.nl/article/122-google-authenticator-instellen

Met Google Authenticator kun je tweestapsverificatie inschakelen om je Mijn KeurigOnline omgeving beter te kunnen beveiligen. Volg de onderstaande stappen om Google Authenticator op je smartphone in te stellen.

### Stap 1: Download Google Authenticator- Gebruik de de **App Store** (iPhone) of **Google Play Store** (Android).- Download en installeer de app.### Stap 2: Twee-stap-verificatie activeren voor je account- Log in op Mijn KeurigOnline en ga naar **Mijn gegevens.**- Zoek naar de optie **Twee-factor-authenticatie**.- Klik op **Stel externe 2FA in**.![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/671628061198281bd860a335/file-fw5kJ1lpB1.png)

### Stap 3: Koppel je account aan Google Authenticator- Nadat je tweestapsverificatie hebt ingeschakeld, verschijnt er een QR-code op het scherm.- Open de **Google Authenticator-app** op je smartphone.- Klik op het **+ teken** rechts onder op je scherm.- Kies voor **Scan QR-code**.- Richt de camera van je telefoon op de QR-code die binnen Mijn KeurigOnline wordt getoond.Nadat je de QR-code hebt gescand, zie je een 6-cijferige code in de Google Authenticator-app. Als titel van de code zal je Mijn KeurigOnline: {je klantnummer}  zien staan.

Vanaf nu, wanneer je inlogt op je Mijn KeurigOnline omgeving, moet je naast je wachtwoord ook de 6-cijferige code invoeren die gegenereerd wordt door de Google Authenticator-app. Deze code wordt elke 30 seconden vernieuwd.
