# Hoe kan ik mijn pakket upgraden?

> Bron: https://help.keurigonline.nl/article/37-hoe-kan-ik-mijn-pakket-upgraden

Je pakket upgraden kan je doen binnen Mijn KeurigOnline.

Zodra je bent ingelogd op het dashboard zie je een overzicht van je actieve diensten.

Klik in de linker balk op &#39;Diensten&#39; -> &#39;Pakketten&#39;. Je krijgt nu een overzicht van alle pakketten.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db2a95173914f806c0b9a/file-mXcTtq6Gva.png)

Klik op het pakket die je wil upgraden op &#39;Upgraden&#39;. Je krijgt nu een aantal opties welke upgrade je uit wil voeren.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db3805940934e140f3af2/file-VSRuziyuVQ.png)

Selecteer de optie die je wil en klik op &#39;Upgrade bestellen&#39;. Je pakket is nu geüpgraded!
