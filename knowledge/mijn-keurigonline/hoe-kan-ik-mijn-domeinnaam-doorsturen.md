# Hoe kan ik mijn domeinnaam doorsturen?

> Bron: https://help.keurigonline.nl/article/61-hoe-kan-ik-mijn-domeinnaam-doorsturen

Het instellen van een doorsturing kan je doen binnen Mijn KeurigOnline.

Zodra je bent ingelogd op het dashboard zie je een overzicht van je actieve diensten. Hier klik je op Mijn domeinen

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/6943de3e9c8f912774281725/file-JVs3eh52wu.png)

Om de doorsturing in te stellen van je domeinnaam klik je op de knop &#39;Beheren&#39; achter de betreffende domeinnaam

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/6943de5b4f7f21406b17eef2/file-6CGcWKuXRF.png)

Je krijgt nu een aantal opties om de domeinnaam te beheren. Selecteer in dit geval &#39;Doorsturingen beheren&#39;.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/6943de66ee7c056afa8112a7/file-Mr8C3ipDFR.png)

Er volgt nu een kolom waarop je de doorsturing in kan stellen. Vul bij &#39;Redirect naar&#39; het volledige pad in, inclusief https://
