# Waarom kan ik geen bestelling plaatsen in Mijn KeurigOnline?

> Bron: https://help.keurigonline.nl/article/73-waarom-kan-ik-geen-bestelling-plaatsen-in-mijn-keurigonline

Bestellingen worden altijd gedaan via de website zelf. Om makkelijk en snel een nieuwe dienst te bestellen klik je binnen Mijn KeurigOnline op &#39;Nieuwe diensten bestellen&#39;
