# Hoe verhuis ik een domein naar KeurigOnline?

> Bron: https://help.keurigonline.nl/article/59-hoe-verhuis-ik-een-domein-naar-keurigonline

Om een domeinnaam naar ons te verhuizen hebben we meestal een EPP-code (verhuiscode) nodig. Deze token kan je opvragen bij de huidige provider. Na de bestelling kan je de token aan ons doorgeven via Mijn KeurigOnline onder het menu-item [Verhuizingen](https://mijn.keurigonline.nl/verhuizingen). Daarmee kan je de verhuizing opstarten. Het is ook altijd mogelijk om de code naar ons te mailen. Dan pakken wij het voor je op.
