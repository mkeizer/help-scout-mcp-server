# Hoe beheer ik mijn DNS records ?

> Bron: https://help.keurigonline.nl/article/76-hoe-beheer-ik-mijn-dns-records

Aanpassingen binnen de DNS kan je doen via Mijn KeurigOnline.

Zodra je bent ingelogd op je dashboard zie je een overzicht van je actieve diensten. Klik hier op Mijn domeinen.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/6943de3e9c8f912774281725/file-JVs3eh52wu.png)

Om de DNS te beheren van je domeinnaam klik je op de knop &#39;Beheren&#39; achter de betreffende domeinnaam

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/6943de5b4f7f21406b17eef2/file-6CGcWKuXRF.png)

Je komt nu in het overzicht met al je DNS records en zit automatisch in het tabje DNS Beheren.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/6943df816b69947dcfc60037/file-NHW6TMjC1p.png)

Je krijgt nu een volledig overzicht van alle bestaande DNS-records. Hier kan je zonder problemen records toevoegen, verwijderen of aanpassen.
