# Hoe kan ik inloggen op de FTP server?

> Bron: https://help.keurigonline.nl/article/77-hoe-kan-ik-inloggen-op-de-ftp-server

Je kunt eenvoudig en snel inloggen op de FTP-server met de DirectAdmin-gegevens die je van ons hebt ontvangen. De hostnaam voor je FTP-verbinding is doorgaans dezelfde als de URL van DirectAdmin. Je kunt ook simpelweg je eigen domeinnaam gebruiken als alternatief.
