# Welke PHP versie draait er op mijn hostingpakket?

> Bron: https://help.keurigonline.nl/article/82-welke-php-versie-draait-er-op-mijn-hostingpakket

Kijk voor je actuele PHP versie in DirectAdmin. Open eerst de Extra Features, er komt nu een dropdown naar beneden met een aantal opties.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664c9533dd0f8c60bb3fbfe7/file-iQruDt8yWH.png)

Klik op Select PHP Version.

Hier kan je je versie zien, aanpassen en de instellingen van je PHP installatie wijzigen.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664c981e1f3fa9421e286df0/file-s0mNcebHuc.png)

Op dit voorbeeld maakt de website gebruik van PHP 7.4. Je kan op de PHP-versie klikken om een andere versie te willen selecteren. Eventuele PHP opties kan je ook aanvinken om deze in gebruik te nemen
