# Hoe restore ik een backup via JetBackup

> Bron: https://help.keurigonline.nl/article/48-hoe-restore-ik-een-backup-via-jetbackup

Om te beginnen met het restoren van een backup moet je zijn ingelogd op DirectAdmin. De DirectAdmin-omgeving is altijd te benaderen via; https://jedomeinnaam.nl:2222

Zodra je bent ingelogd op DirectAdmin krijg je een navigatiebalk te zien. Klik op Extra Programma&#39;s.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664dad65d593d719515c97ac/file-tKDWTNy0d8.png)

In de opties die je nu te zien krijgt klik je op JetBackup

Het dashboard van JetBackup zal nu eerst even moeten laden en toont je daarna een overzicht met de opties om te herstellen. Bedenk hierbij goed wat je terug wil zetten.

Wil je bijvoorbeeld enkel een maibox herstellen, kies dan voor Email Accounts. Het is in dat geval niet verstandig om een &#39;Full Backup&#39; te herstellen omdat hiermee een heleboel onnodige data wordt overschreven.

![](https://static.keurigonline.nl/images/helpdesk/FAQ/jetbackuprestore_stap2.png)

Nadat je een van de opties hebt geselecteerd kom je in de lijst waar je mogelijke backups worden getoond. Hieruit kan je dus kiezen bij het herstellen.

![](https://static.keurigonline.nl/images/helpdesk/FAQ/jetbackuprestore_stap3.png)

Je selecteert hier dus 1 van de backups die je wil gaan herstellen. Zodra je deze hebt geselecteerd wordt de rij van de backup oranje.

Als je het zeker weet gaan we daad bij onze woorden voegen, je mag klikken op &#39;Restore&#39;

![](https://static.keurigonline.nl/images/helpdesk/FAQ/jetbackuprestore_stap4.png)

Nadat je op &#39;Restore&#39; hebt geklikt is er nog 1 belangrijk vinkje die ingeschakeld moet worden. Deze optie zorgt ervoor dat je backup wordt &#39;samengevoegd&#39; met je bestaande data.

Als je dit vinkje dus niet aanvinkt verdwijnt je oude en/of nieuw data en gaat dit dus volledig terug naar het punt van de backup. Eigenlijk is dit **nooit **aan te raden.

![](https://static.keurigonline.nl/images/helpdesk/FAQ/jetbackuprestore_stap5.png)

Nadat je dit hebt gedaan zal je naar de &#39;wachtrij&#39; worden gestuurd waar je de status van je restore kan inzien.

Zodra deze zegt &#39;Completed&#39; dan heb je succesvol je backup terug gezet! Goed bezig!
