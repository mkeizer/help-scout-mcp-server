# Waarom is mijn WordPress website langzaam?

> Bron: https://help.keurigonline.nl/article/103-waarom-is-mijn-wordpress-website-langzaam

De snelheid van de website kan afhangen van onze server of van de installatie. Als je het vermoeden heeft dat het aan de installatie ligt, bijvoorbeeld omdat andere domeinen van jou wel snel werken, dan kan je WordPress onder de loep nemen met een plugin genaamd ["Query Monitor"](https://wordpress.org/plugins/query-monitor/). Met deze plugin kan je heel snel achterhalen welke plugin, of onderdeel van het thema, voor de vertraging zorgt.

Je kan ook eerst zelf proberen om wat plug-ins uit te schakelen en van thema te wisselen om te kijken of de website sneller wordt.

Als je er van overtuigd bent dat het niet aan de installatie ligt maar aan onze servers, dan kan je contact zoeken met onze helpdesk.
