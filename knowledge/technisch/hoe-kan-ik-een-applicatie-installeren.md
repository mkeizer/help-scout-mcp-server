# Hoe kan ik een applicatie installeren?

> Bron: https://help.keurigonline.nl/article/53-hoe-kan-ik-een-applicatie-installeren

Om een applicatie te installeren in je hostingpakket moet je zijn ingelogd op DirectAdmin. De DirectAdmin-omgeving is altijd te benaderen via; https://jedomeinnaam.nl:2222

Zodra je bent ingelogd op DirectAdmin krijg je een navigatiebalk te zien. Klik op &#39;Aanbevolen Applicaties&#39;

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db8f6e1989867dcefce35/file-YEYssX78UB.png)

Je krijgt hierin een dropdown met een aantal opties. In dit voorbeeld klikken we op WordPress om deze te installeren.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db90f5173914f806c0ba0/file-WSLUF2yYBs.png)

Je krijgt nu een overzicht van de applicatie. Klik om de applicatie te installeren op &#39;Installeer deze applicatie. Mocht je geen gebruik willen maken van WordPress, zoek dan bovenin bij &#39;Zoek naar een app&#39; naar de applicatie die jij graag wil installeren.

Je krijgt nu een flink aantal opties welke je naar eigen wens in kan vullen. Nadat je alles hebt ingevuld, selecteer je onderaan de pagina &#39;Installeren&#39;. Je hebt nu succesvol een applicatie geinstalleerd.
