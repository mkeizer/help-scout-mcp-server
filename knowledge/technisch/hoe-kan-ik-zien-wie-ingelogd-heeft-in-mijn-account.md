# Hoe kan ik zien wie ingelogd heeft in mijn account?

> Bron: https://help.keurigonline.nl/article/43-hoe-kan-ik-zien-wie-ingelogd-heeft-in-mijn-account

Om de inloghistorie van je hostingpakket in te zien moet je zijn ingelogd op DirectAdmin. De DirectAdmin-omgeving is altijd te benaderen via; https://jedomeinnaam.nl:2222

Zodra je bent ingelogd op DirectAdmin krijg je een navigatiebalk te zien. Klik op &#39;Systeem Info & Bestanden&#39;

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664dbb8fd593d719515c97bf/file-jOG5fJynJw.png)

Klik in de opties die je krijgt op &#39;Siteoverzicht / Statistieken / Logs&#39;

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664dbbc1d593d719515c97c0/file-9yV8fot5ao.png)

Je krijgt nu een overzicht van alle domeinnamen. Rechtsbovenin het scherm heb je de optie &#39;Aanmelding Geschieden&#39;. Klik deze aan.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664dbc935940934e140f3afd/file-0XI8vyzzgE.png)

Je krijgt nu een overzicht van de datums dat er is ingelogd met het bijbehorende IP-adres van degene die is ingelogd.
