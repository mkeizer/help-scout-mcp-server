# Hoe maak ik een MySQL database aan?

> Bron: https://help.keurigonline.nl/article/55-hoe-maak-ik-een-mysql-database-aan

Om te beginnen met het aanmaken van een database moet je zijn ingelogd op DirectAdmin. De DirectAdmin-omgeving is altijd te benaderen via; https://jedomeinnaam.nl:2222

Zodra je bent ingelogd op DirectAdmin krijg je een navigatiebalk te zien. Klik op Accountbeheer.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664daab25940934e140f3aea/file-HZ892C8FTr.png)

Uit alle mogelijkheden die je krijgt mag je klikken op &#39;Databasen&#39;.

Je krijgt nu een uitgebreid overzicht van je (eventuele) bestaande databasen.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664dab0d1f3fa9421e286eb9/file-2YSyJF71bI.png)

Om snel een database aan te maken, kan je bij &#39;Database Naam&#39; een naam invoeren. Dit is een naam die je zelf mag bedenken. Indien het je eerste installatie van WordPress is kan dit bijvoorbeeld zijn wp1.

Na het invoeren van de naam klik je op &#39;Aanmaken&#39;

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664dabcddd0f8c60bb3fc0cf/file-tdzON7kRWN.png)

Als de database succesvol is aangemaakt zie je de databasenaam, je gebruikersnaam en wachtwoord.

Deze gegevens kan je ook gebruiken om in te loggen op [phpMyAdmin](https://helpdesk.keurigonline.nl/article/79-hoe-kan-ik-inloggen-in-phpmyadmin).

Je hebt nu succesvol een MySQL database aangemaakt.
