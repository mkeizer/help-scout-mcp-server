# Hoe maak ik een extra FTP account aan?

> Bron: https://help.keurigonline.nl/article/52-hoe-maak-ik-een-extra-ftp-account-aan

Om te extra FTP account aan te maken moet je zijn ingelogd op DirectAdmin. De DirectAdmin-omgeving is altijd te benaderen via; https://jedomeinnaam.nl:2222

Zodra je bent ingelogd op DirectAdmin krijg je een navigatiebalk te zien. Klik op Accountbeheer.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664daf4e5940934e140f3af1/file-8mwrCKTwyM.png)

Klik in de opties die je krijgt op FTP beheer. Je krijgt nu een overzicht van alle huidige FTP-accounts.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664daf79e1989867dcefce2a/file-AcpSdM464f.png)

Om een extra FTP account aan te maken klik je bovenin op &#39;FTP account Aanmaken&#39;.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664dafa4dd0f8c60bb3fc0d3/file-uRlWK590EH.png)

Je kan nu een gebruikersnaam invullen met een bijbehorend wachtwoord. Ook kan je specifiek instellen waar het FTP-account toegang tot heeft. Als je alles hebt ingesteld klik je op &#39;Aanmaken&#39;.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664dafed1f3fa9421e286ec3/file-eM90ia5FMA.png)

Als het FTP-account succesvol is ingesteld krijg je de inloggegevens te zien. Deze kan je nu gebruiken om in te loggen op je FTP programma.
