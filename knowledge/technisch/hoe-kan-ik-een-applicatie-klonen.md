# Hoe kan ik een applicatie klonen?

> Bron: https://help.keurigonline.nl/article/54-hoe-kan-ik-een-applicatie-klonen

Om een applicatie te klonen in je hostingpakket moet je zijn ingelogd op DirectAdmin. De DirectAdmin-omgeving is altijd te benaderen via; https://jedomeinnaam.nl:2222

Zodra je bent ingelogd op DirectAdmin krijg je een navigatiebalk te zien. Klik op Mijn Applicaties.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db707d593d719515c97b9/file-bgvkFNGp3N.png)

Je krijgt nu een lijst met je applicaties te zien. Klik de applicatie aan die je wil klonen.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db7781f3fa9421e286ec7/file-QyZitdB6Bt.png)

Je krijgt een flink aantal opties. Het gaat er in dit geval om klonen. Klik op het icoontje die blauw omcirkeld is. 

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db8105940934e140f3af9/file-wk0h8R0Eer.png)

Nadat je dit hebt aangeklikt krijg je opties waar je een kloon naar wil maken. Selecteer bij bestemming de domeinnaam waar je de kloon heen wil hebben. Optioneel kan je ook een pad invullen, om de website bijvoorbeeld te klonen naar /test of /oud

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db88fe1989867dcefce34/file-sykhLAlY3Z.png)

Indien alles staat zoals je dat wil, klik je onderin op &#39;Klonen&#39;. Je hebt nu succesvol van de website gemaakt.
