# Kan ik gebruik maken van Redis?

> Bron: https://help.keurigonline.nl/article/119-kan-ik-gebruik-maken-van-redis

Bij het optimaliseren van je applicatie kan Redis Cache een belangrijke rol spelen. Redis biedt een snelle en efficiënte manier om gegevens in het geheugen op te slaan, waardoor je de laadtijden van je applicatie kunt verkorten en de belasting op je database kunt verminderen. Dit zorgt ervoor dat veelgebruikte gegevens sneller beschikbaar zijn, wat de gebruikerservaring verbetert.

In ons pakket is Redis een optionele functie. Het kan echter zijn dat Redis nog niet is ingeschakeld voor jouw omgeving. Als je deze caching-oplossing wilt gebruiken, kun je contact opnemen met onze helpdesk. We helpen je graag met het activeren van Redis. We bieden op dit moment geen ondersteuning voor memcached.
