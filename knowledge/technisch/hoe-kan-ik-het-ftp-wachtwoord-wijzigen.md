# Hoe kan ik het FTP wachtwoord wijzigen?

> Bron: https://help.keurigonline.nl/article/51-hoe-kan-ik-het-ftp-wachtwoord-wijzigen

Om te extra FTP account aan te maken moet je zijn ingelogd op DirectAdmin. De DirectAdmin-omgeving is altijd te benaderen via; https://jedomeinnaam.nl:2222

Zodra je bent ingelogd op DirectAdmin krijg je een navigatiebalk te zien. Klik op Accountbeheer.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664daf4e5940934e140f3af1/file-8mwrCKTwyM.png)

Klik in de opties die je krijgt op FTP beheer. Je krijgt nu een overzicht van alle huidige FTP-accounts.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db0505173914f806c0b99/file-RiPxf24Gpl.png)

Klik voor het FTP account waar je het wachtwoord van aan wil passen op het potloodje. Je krijgt nu een vervolgscherm waar je het wachtwoord opnieuw in kan voeren.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5acb572d0428630750923499/images/664db093dd0f8c60bb3fc0d5/file-Coed5BiXHF.png)

Voer een nieuw wachtwoord in en klik op Bewerken. Je hebt nu het wachtwoord succesvol aangepast.
